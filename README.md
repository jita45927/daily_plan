# 每日计划 (Daily Plan)

一款基于 Tauri 2 和 Vue 3 开发的跨平台任务管理与桌面整理工具。

## 功能特性

### 任务管理
- ✅ 添加、编辑、删除任务
- ✅ 任务状态切换（完成/未完成）
- ✅ 任务颜色标记和加粗显示
- ✅ 任务排序和拖拽重排
- ✅ 任务回收站（支持恢复和永久删除）

### 计时器功能
- ⏱️ 倒计时功能（设定分钟数）
- ⏱️ 定时提醒功能（设定具体时间）
- ⏱️ 计时器数据持久化
- ⏱️ 程序重启后自动校准计时状态
- ⏱️ 计时完成提醒和音效

### 桌面整理
- 🧹 自动整理桌面文件到分类文件夹
- 🧹 支持分类：可执行文件、图片文件、其他文件、压缩包
- 🧹 桌面重复文件检测和清理
- 🧹 文件夹分析窗口（查看分类统计）

### 文件夹整理
- 📂 自定义文件夹整理（默认下载目录）
- 📂 文件夹重复文件检测和清理
- 📂 分析指定文件夹下的子文件夹

### 窗口特性
- 🪟 窗口拖拽移动
- 🪟 窗口贴边吸附（自动对齐屏幕边缘）
- 🪟 窗口锁定和折叠
- 🪟 右键菜单支持
- 🪟 系统托盘图标

### 启动体验
- 🎨 欢迎画面（带进度条）
- 🎨 进度条实时显示加载进度
- 🎨 主窗口加载完成后自动淡出

## 技术栈

### 前端
- **框架**: Vue 3.5+ (Composition API)
- **状态管理**: Pinia 4.0+
- **样式**: Tailwind CSS 3.4+
- **构建工具**: Vite 5.4+
- **语言**: TypeScript 5.4+

### 后端
- **框架**: Tauri 2.0+
- **语言**: Rust 1.70+
- **数据库**: SQLite (rusqlite)
- **文件操作**: Windows API
- **图标**: Tray Icon

## 安装与运行

### 前置依赖
- **Node.js**: 18+
- **Rust**: 1.70+
- **Tauri CLI**: 2.0+

### 开发模式

```bash
# 安装前端依赖
npm install

# 启动开发服务器
npm run tauri dev
```

### 构建生产版本

```bash
# 构建前端
npm run build

# 构建Tauri应用
npm run tauri build
```

## 使用说明

### 任务管理
1. 在输入框中输入任务内容，按回车添加
2. 点击任务前的复选框标记完成/未完成
3. 右键点击任务查看更多操作（编辑、删除、计时器）
4. 任务完成后可移动到回收站

### 计时器
1. 右键点击任务，选择"设置倒计时"或"设置定时"
2. 设置倒计时分钟数或定时时间
3. 计时器会自动保存，程序重启后继续计时

### 桌面整理
1. 点击左侧菜单"整理桌面"
2. 程序会自动将桌面文件分类整理到对应文件夹
3. 可查看分析窗口了解文件分布情况

### 文件夹整理
1. 点击左侧菜单"整理文件夹"
2. 默认整理系统下载目录，也可选择其他文件夹
3. 支持检测和清理重复文件

## 配置说明

### Tauri配置
- 窗口尺寸：可拖拽调整
- 窗口位置：自动保存上次位置
- 贴边吸附：自动对齐屏幕边缘
- 系统托盘：最小化时显示托盘图标

### 数据库配置
- 数据库位置：用户应用数据目录
- 自动备份：任务数据自动保存
- 回收站保留：已删除任务保留30天
